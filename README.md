# Customer-Churn-Prdiction Syste
Overview

Customer Churn Prediction is a Machine Learning project designed to identify customers who are likely to discontinue a service. By analyzing customer behavior, billing information, and service usage patterns, the system helps businesses improve customer retention and reduce revenue loss.

Key Features
Predict customer churn using Machine Learning
Data preprocessing and feature engineering
Model training and performance evaluation
Customer behavior analysis and visualization
Scalable and deployment-ready architecture
Tech Stack
Language: Python
Libraries: Pandas, NumPy, Scikit-learn, Matplotlib
Environment: Jupyter Notebook
Dataset Information

The dataset contains:

Customer demographics
Account and billing details
Subscription information
Service usage patterns
Churn status

Target Variable: Churn (Yes / No)

Machine Learning Models
Logistic Regression
Random Forest
Support Vector Machine (SVM)
Decision Tree Classifier
